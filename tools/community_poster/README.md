# 리치킷 (ReachKit)

커뮤니티 홍보 루틴 도우미 — **사이트 분석 · 문구 · 채널 · 작성 보조 · 가드레일**.

특정 서비스에 종속되지 않습니다. 사용자가 넣은 **제품 URL** 기준입니다.

**판매 준비** · 현재 **성과 검증** 단계.

## 실행

```powershell
# 권장
C:\Users\hysoo\Projects\RoadLog\tools\community_poster\ReachKit.bat

# 또는
cd C:\Users\hysoo\Projects\RoadLog
.\.venv\Scripts\python.exe tools\community_poster\app.py
```

바탕화면 바로가기:

```powershell
.\.venv\Scripts\python.exe tools\community_poster\assets\_make_ko_shortcut.py
```

## 빠른 흐름

1. 개요 → **내 사이트 주소** 입력 → 분석 + 홍보글  
2. 작성 → **커뮤니티 주소** 입력 (가입·캡차·휴대폰 인증은 사용자 1회)  
3. 구조 파악 → 게시판 찾기 → 스마트 작성 (등록은 직접)  
4. 검증 · 성공률 50%+ / 시도 5+
