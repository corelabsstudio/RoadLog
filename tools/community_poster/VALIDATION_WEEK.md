# 이번 주 성과 검증 — 채널 3곳 고정

**제품:** 리치킷 (리치킷)  
**홍보 대상:** 사용자가 입력한 사이트 주소  
**원칙:** 이 3곳만 · 채널당 주 1~2회 · **등록은 직접** · 매번 문구 다른 버전

---

## 왜 이 3곳인가

| # | 채널 | 검증하는 것 | 관점 |
|---|------|-------------|------|
| 1 | **네이버 블로그** | 사이트 분석→문구→글쓰기 주소 작성 보조 | 검색·유입 |
| 2 | **카카오 오픈채팅** | 가이드형(복붙) 루틴·전환 | 전환율 |
| 3 | **링크드인** | 짧은 포스트 작성 보조 | 기업·팀 도입 |

다른 카페·디시 등은 **이번 주 제외** (성공률 측정이 흐려짐).

## 주간 스케줄 (예시)

| 요일 | 채널 | 할 일 | 예상 시간 |
|------|------|--------|-----------|
| **월** | 네이버 블로그 | 분석+사용 방법형 글 1편 · 등록 직접 | 20분 |
| **화** | 오픈채팅 | 짧은 버전 1방 · 링크 1개 | 10분 |
| **수** | 링크드인 | 짧은 포스트 1 · 등록 직접 | 10분 |
| **목** | 네이버 블로그 | **다른 버전**으로 2편째 또는 수정 | 15분 |
| **금** | 오픈채팅 또는 링크드인 | 반응 팔로업 / 짧은 1회 | 10분 |
| 주말 | — | **검증 대시보드** 정리 · 전환 메모 | 10분 |

**주간 시도 목표:** 5회 이상  
**성공률 목표:** 50%+  
**전환:** 가입·문의 있으면 대시보드에 메모

## 채널별 체크리스트

### 1) 네이버 블로그 (월·목)

1. 리치킷 → 내 제품 URL 설정 → 사이트 분석 + 홍보글  
2. 스타일 **사용 방법형** · 필요 시 다른 버전  
3. **이번 주 검증 3채널** → 네이버 블로그 적용  
4. 로그인 바로 연결 → 작성 (등록 직접)  
5. 제목 키워드는 내 제품에 맞게  
6. 링크는 본문 **하단 1개**만  

**성공:** 발행됨 또는 에디터에 제목·본문 입력 성공  

### 2) 카카오 오픈채팅 (화·금)

1. 스타일 **짧은 버전** 생성  
2. 채널 적용 → open.kakao.com (자동 로그인은 기대하지 않음)  
3. 방 규칙 확인 후 **직접 붙여넣기**  
4. 인사 + 제품 링크 1개 · 가격은 질문 오면  

**성공:** 메시지 전송 완료 (툴은 문구·루틴 검증)  
**전환 시:** 검증 대시보드 → 전환 메모  

### 3) 링크드인 (수·금)

1. **짧은 버전** + 필요 시 「팀 도입 문의 환영」  
2. 이번 주 3채널 → 링크드인 적용  
3. 로그인 → 작성 화면 입력 → **게시 직접**  
4. (보너스) 프로필 소개에 제품 한 줄  

**성공:** 포스트 게시 또는 작성란 입력 성공

## 금지 (이번 주)

- 위 3곳 외 카페 10곳 도배  
- 같은 본문 그대로 재사용 (가드레일이 막음 → **다른 버전 만들기**)  
- 등록 자동 클릭 ON  
- 허위 후기

## 주말 리뷰 (Go / No-Go)

검증 대시보드에서:

| 질문 | Go에 가깝다 |
|------|-------------|
| 시도 ≥ 5? | 예 |
| 성공률 ≥ 50%? | 예 |
| 3채널 모두 최소 1회? | 예 |
| 자발적으로 쓰고 싶은가? | 예 |
| 전환 메모 또는 체감 반응? | 있으면 가산 |

No-Go면: 실패한 채널만 원인 메모(로그인/글쓰기/규정) 후 다음 주 같은 3곳으로 재시도.
